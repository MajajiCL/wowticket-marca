# WowTicket — Manual de Marca

Rediseño premium de **brand.wowticket.cl** (Next.js 15 + React 19) **manteniendo el contenido original**:
uso del logo en 3 fondos, color oficial `#e8217c` (Pantone P 71-8), pack de descarga y ecosistema
(Wowticket / WowPartner / Embajadores). Tipografías Montserrat + Outfit.

## 🚀 Desarrollo
```bash
npm install
npm run dev      # http://localhost:3050
npm run build && npm start   # producción
```

## ☁️ Despliegue
App Next.js estática → **Vercel** / Netlify / hosting Node. Destino: `brand.wowticket.cl`.

Logos oficiales en `public/logos/` (color, blanco, negro). Pack completo: `assets-wowticket-v3.zip` (enlace oficial en la web).
